Keyboard Shortcuts

Numpad + Toggle speed ON/OFF
` Reset to 1x speed
] Increase speed +0.5x
[ Decrease speed -0.5x
Shift + ] Fine increase +0.1x
Shift + [ Fine decrease -0.1x

All shortcuts are fully customizable in the extension settings.

Installation

1

Download the Extension

Get the latest release from GitHub and extract the ZIP file.

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your browser.

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner.

4

Load the Extension

Click "Load unpacked" and select the extracted folder.

5

Enjoy!

Go to YouTube and start controlling your playback speed like a pro.
